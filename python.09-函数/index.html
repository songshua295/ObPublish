    • # 参数传递

    • 两种传递方式
      1. 直接把实参依次传递进去 fun(a,b)
      2. 使用关键字指定传的值fun(b=12,a=23)
    • 参数传递的内存分析
    • 如果形参是不可变对象，那么函数不会对实参产生影响
    • 如果实参是可变对象，那么函数会影响具体的实参对象

      def fun(nums):
    odd=[]
    even=[] #偶数
    for i in nums:
        if i%2:
            odd.append(i)
        else:
            even.append(i)
    return odd,even

    • # 参数定义-个数可变

    • 主要用在形参参数可变的情况，当形参的个数位置的时候，一般使用可变位置参数，当参数是关键字形式的时候，就使用关键字可变参数
      1. 个数可变的位置参数：使用*name一个星号定义，结果是一个元组
      2. 个数可变的关键字参数 **name：使用两个星号定义，结果是一个字典
    • 注意：参数列表只能有一个可变位置参数，也只能有一个可变关键字参数。当两个可变参数都有一个的时候，可变位置参数在前
    • 例子：print函数
