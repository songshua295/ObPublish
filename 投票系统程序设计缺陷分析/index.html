# 投票系统程序设计缺陷分析

🔗MetaURL: https://www.mozhe.cn/bug/detail/anBmazJ1NWIvcnZmcldxMnUvdzZKQT09bW96aGUmozhe

# 背景介绍

年终了，公司组织了各单位"文明窗口"评选网上投票通知。

# 实训目标

1、了解浏览器插件的使用； 2、了解开发程序员对IP地址获取方式； 3、了解网络协议软件的使用，如burpsuite等；

# 解题方向

  1. 点击ggg投票进行抓包

  1. 添加X-Forwarded-For字段

    HTTP X-Forwarded-For用来识别通过HTTP代理或负载均衡方式连接到Web服务器的客户端最原始的IP地址的HTTP请求头字段。

    简单来说就是返回你的ip地址

    1

    X-Forwarded-For:192.168.1.1

  2. ctrl+i将添加到 intruder （攻击者）中，可以修改ip为可变化的值，快速的返回请求

    Cluster bomb 集速炸弹 下方的特殊符号用于修改的值

设置特殊符号中的值的变化，从1加到255 （因为ip地址单个数值最大255）

# 遇到的问题

💡 burp suite会弹出错误，当使用集束炸弹的时候需要分别对两个值分别进行设置numbers设置。当出现空值错误的时候，可能是因为burpsuite内的bug，点击hex和十进制就ok
