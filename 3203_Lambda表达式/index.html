# 3203_Lambda表达式

只能简化函数式接口的匿名内部类的写法

  1. 必须是接口，有且只有一个抽象方法
  2. 一般这个接口上会加函数式接口的注解@FunctionalInterface

作用

简化匿名内部类的代码写法（Java8的一个新特性)

# 书写格式

1
2
3

(匿名内部类的被重写的形参列表)->{
	被重写的方法体;
};

# 书写格式进一步简化

  1. 括号的形参类型可以不写
  2. 如果只有一个参数，参数类型可以省略，同时()可以省略
  3. 如果Lambda的方法体只有一行代码，可以省略大括号不写，同时需要省略分号；
  4. 如果只有一行代码，且是return语句，可以省略return不写，分号也需要省略

package LambdaTest;  
  
import java.security.PublicKey;  
import java.sql.SQLOutput;  
  
public class Test01 {  
    public static void main(String[] args) {  
        String name = "张三";  
        // 采用匿名内部类  
        Swimming swim1 = new Swimming() {  
            @Override  
            public void swim() {  
                System.out.println("swimming");  
            }  
        };  
        swim1.swim();  
  
        //    采用Lambda表达式简化格式  
        Swimming swim2 = () -> {  
            System.out.println(name + "在游泳");  
        };  
        swim2.swim();  
    //        采用lambda表格式2  
        go(()->{  
            System.out.println("在河边游泳");  
        });  
  
    }  
    public  static  void go(Swimming s){  
        System.out.println("start---");  
        s.swim();  
        System.out.println("end-----");  
    }  
}  
  
interface Swimming {  
    void swim();  
}
