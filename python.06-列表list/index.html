  2. `[start:end:step]`切片：从列表中获取一个片段的列表，拷贝成一个新的列表对象。id值和原来的不一样。**切片会产生一个新的列表对象**
  
      切片的start和end中，不会包含较小值索引那个值，如下面[1:5:2]例子所示
  
      ```python
      a=[1,2,3,4,5,6,7]
# 切片
    print(a[1:5:2])
# 步长为负数
    print(a[5:1:-1])
    #省略参数
    print(a[:4:2])
    ```
输出结果

  ```python
  [2, 4]
  [6, 5, 4, 3]
  [1, 3]

      a=[1,2,3,4,5,6,7]
  b=["a","b","c","d","e"]
  # append
  a.append("你好")
  print(a)
  a.append(b)
  print(a)
  
  # extend
  a=[1,2,3,4,5,6,7]
  b=["a","b","c","d","e"]
  a.extend("个")
  print(a)
  a.extend(b)
  print(a)
  
  # insert
  a=[1,2,3,4,5,6,7]
  b=["a","b","c","d","e"]
  a.insert(3,b)
  a.insert(4,"insert")
  print(a)
  
  # 切片添加
  a=[1,2,3,4,5,6,7]
  b=["a","b","c","d","e"]
  a[1::]=b
  print(a)
